QMK corne keyboard - keymap dvorak configuration


- Build:
from this route /Users/<username>/qmk_firmware/keyboards/crkbd/keymaps/carloslafuente
qmk compile -kb crkbd -km carloslafuente

- Tool to flash corne keyboard:
qmk toolbox
